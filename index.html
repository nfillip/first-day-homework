Common Terminal/Bash Commands

  • cd <FOLDERNAME> (changes folder)

  • cd ~ (changes to home folder)

  • cd .. (moves up one folder)

  • ls (lists files/directories in current folder)

  • ls -a (lists all files/directories; includes hidden files)

  • pwd (shows current file path)

  • mkdir <FOLDERNAME> (creates a new folder)

  • touch <FILENAME> (creates a file)

  • rm <FILENAME> (deletes a file)

  • rm -r <FOLDERNAME> (deletes a folder, note the -r)

  • mv <FILENAME> <DESTINATION> (moves a file)

  • mv <OLDFILENAME> <NEWFILENAME> (renames a file)

  • cp <FILENAME> <DESTINATION> (copies a file)

  • open . (opens the current folder, TERMINAL SPECIFIC)

  • open <FILENAME> (opens a specific file, TERMINAL SPECIFIC)

  • explorer . (opens the current folder, BASH SPECIFIC)

  • explorer <FILENAME> (opens the specific file, BASH SPECIFIC)

  • code . (opens the current folder in VSCode; Mac users may have to run an extra command to get this set up)
